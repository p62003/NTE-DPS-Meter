# NTE DPS Meter

Neverness to Everness (NTE) 实时伤害统计工具。

通过被动封包分析读取战斗数据，不注入游戏内存、不修改游戏文件。

**版本**: v2.00 | **适用游戏版本**: v1.08+

## 功能

### DPS 计算方式

| 项目 | 说明 |
|------|------|
| 计算基准 | 根据统计时间计算平均每秒伤害 |
| 计算范围 | 个人该队伍全角色 |
| 计算起点 | 首击后启动 |
| 计算终点 | 末击为止 |

真实反映战斗过程中的所有操作 — 死亡、发呆、走位失误皆会因计时持续但未造成伤害而导致 DPS 下降。

### 免费功能

- **DPS 浮窗** — 置顶透明卡片，圆形角色头像 + 呼吸光环 + 三行实时数据，战斗时自动展开、脱战自动收回
- **角色识别** — 自动识别当前操作角色，支持全 17 角色
- **战报管理** — 重置时自动存为战报，可依伤害范围筛选、排序、上传至社群
- **社群排行榜** — 各角色伤害排名（依战斗时长分 Tier），需上传战报后解锁
- **快捷键** — 显示/隐藏浮窗、重置统计（F6），可自定义组合键
- **网卡选择** — 流量不明时可手动选择网卡，持续支持加速器 ExitLag
- **三语系界面** — 繁体中文 / 简体中文 / English（即时切换）
- **系统托盘常驻** — 不占任务栏
- **自动更新** — 手动检查后自动完成更新流程

### 赞助者功能

包含所有免费功能，额外解锁：

- **主窗口** — 更直观的界面，随时查看全队各角色伤害贡献占比，数据保存在本地
- **战报对比** — A/B 并排对比不同队伍配置的输出差异 + 效率摘要
- **编队组合统计** — 社群角色配对采用率热力图（[ntedpsmeter.com](https://ntedpsmeter.com)）
- **分析平台完整存取** — Rankings + Compositions 全功能

### 社群分析平台

线上平台 [ntedpsmeter.com](https://ntedpsmeter.com)，汇集社群战斗数据：

- **角色伤害排行榜** — Tier 分级（1min / 3min / 6min / 9min），显示中位数 + 最大值 + 区间
- **编队配对矩阵** — 可视化角色搭配采用率（赞助者限定）
- **Connect 流程** — 贴上 Device ID 或 Analytics Token 即可在网页查看个人数据
- **三语网站** — EN / 繁中 / 简中
- **隐私友善** — 上传前须明确同意，数据匿名化

## 系统需求

- Windows 10 / 11
- [Npcap](https://npcap.com/) — 封包捕获驱动（安装时勾选 WinPcap 兼容模式）
- 以**管理员身份**运行

## 使用方式

1. 安装 Npcap（首次安装会由更新器自动处理）
2. 运行 `NTE_DPS_Meter.exe`（右键 → 以管理员身份运行）
3. 启动游戏，进入战斗后浮窗自动显示 DPS
4. 从右键菜单可打开主窗口、选择网卡、战报管理、设置快捷键等多项功能

### 默认快捷键

| 快捷键 | 功能 |
|--------|------|
| `F6` | 重置统计并存档战报 |
| `Alt + D` | 显示/隐藏浮窗 |
| `Alt + E` | 显示/隐藏主窗口（赞助者） |

快捷键可在设置中自定义。

## 安全性说明

本工具仅被动读取网络封包，原理等同 Wireshark：

- 不读取、不修改游戏内存
- 不注入 DLL 或 hook 游戏进程
- 不发送任何封包至游戏服务器
- 不修改任何游戏文件

## 常见问题

**Q: 浮窗没有反应？**
A: 确认已以管理员身份运行，且 Npcap 已正确安装。

**Q: 开了加速器后抓不到封包？**
A: 从系统托盘选择「选择网卡」，手动切换到正确的网络接口。

**Q: 显示的伤害数值准确吗？**
A: 数值直接来自游戏客户端发送的战斗封包，与游戏内部计算一致。

**Q: 为什么一直跳防毒检测？**
A: 本程序尚未购买 EV 代码签名证书（价格高昂），Windows Defender 可能标记为未知程序。请将程序所在文件夹加入排除列表：

桌面右键 → 显示设置 → 隐私和安全性 → Windows 安全中心 → 病毒和威胁防护 → 管理设置 → 向下滚动找到「排除项」→ 添加排除项 → 添加排除范围 → 文件夹 → 选择程序所在目录
