# NTE DPS Meter

Neverness to Everness (NTE) 即時傷害統計工具。

透過被動封包分析讀取戰鬥數據，不注入遊戲記憶體、不修改遊戲檔案。

適用遊戲版本: v1.08 (2026-05-04)

## 功能

### DPS 計算方式

| 項目 | 說明 |
|------|------|
| 計算基準 | 根據統計時間計算平均每秒傷害 |
| 計算範圍 | 個人該隊伍全角色 |
| 計算起點 | 首擊後啟動 |
| 計算終點 | 末擊為止 |

真實反映戰鬥過程中的所有操作 — 死亡、發呆、走位失誤皆會因計時持續但未造成傷害而導致 DPS 下降。

### 免費

- **DPS 浮窗** — 置頂透明卡片，圓形角色頭像 + 呼吸光環 + 三行即時數據，戰鬥時自動展開、脫戰自動收回
- **角色辨識** — 自動識別當前操作角色，支援全 17 角色
- **戰報管理** — 重置時自動存為戰報，可依傷害範圍篩選、排序、上傳至社群
- **社群排行榜** — 各角色傷害排名（依戰鬥時長分 Tier），需上傳戰報後解鎖
- **快捷鍵** — 顯示/隱藏浮窗、重置統計（F6），可自訂組合鍵
- **網卡選擇** — 支援加速器、VPN 等多網卡環境（目前持續支援 ExitLag 穩定運行，其餘品牌加速器/VPN 視贊助情況而定）
- **三語系介面** — 繁體中文 / 简体中文 / English（即時切換）
- **系統匣常駐** — 不佔工作列
- **自動更新** — 手動檢查後自動完成更新流程

### Pro（贊助者功能）

包含所有免費功能，額外解鎖：

- **主視窗** — 更直觀的介面，隨時查看全隊各角色傷害貢獻佔比，資料保存在本地
- **戰報系統** — 本機存取戰鬥記錄，隨時回顧
- **戰報比對** —  A/B 並排對比不同隊伍配置的輸出差異 + 效率摘要
- **編隊組合統計** — 社群角色配對採用率熱力圖（[ntedpsmeter.com](https://ntedpsmeter.com)）
- **分析平台完整存取** — Rankings + Compositions 全功能
- **持續優化 UI/UX**
- 🔧 **角色切換次數**（開發中）— 統計切換角色的次數，掌握操作習慣
- 🔧 **角色傷害時間軸**（開發中）— 以圖像顯示各角色在時間軸中的參與度
- 🔧 **戰鬥節奏心跳圖 ECG**（開發中）— 重疊顯示怪物與玩家傷害峰值的時間軸變化，精進操作手法

### DPS 浮窗

| 操作 | 說明 |
|------|------|
| 自動展開 | 戰鬥時偵測到命中後自動從待機圓展開為卡片，顯示角色頭像 + 傷害/命中/DPS |
| 自動收回 | 5 秒無命中自動收回為待機圓 |
| 左鍵點擊（待機時） | 手動展開預覽卡片（不需要戰鬥） |

### 主視窗（Sponsor）

主視窗需 Sponsor 身分才能開啟。可透過右鍵選單、快捷鍵 Alt+E 或雙擊系統匣圖標進入。

**標題列按鈕：**

| 按鈕 | 說明 |
|------|------|
| 重置 | 清除所有戰鬥數據，重新開始計算（同時自動存檔當前戰報） |
| ! | 顯示程式資訊（版本、作者、Discord） |
| + / − | 切換比對模式 |
| 語系圖標 | 切換語系（同右鍵選單） |
| Discord 圖標 | 開啟 Discord |
| ⋮ | 開啟選單（同右鍵選單） |

**排名欄（左）：** 各角色傷害排名、佔比長條圖，點擊角色切換右側細節

**細節欄（右）：** 選中角色的每筆命中記錄與數值

**底部戰報列：**

| 操作 | 說明 |
|------|------|
| 存檔 | 儲存當前戰鬥數據為戰報 JSON |
| 下拉選單 | 切換「即時」或歷史戰報。選擇歷史戰報後主視窗進入回顧模式 |

### 戰報比對模式

從主視窗標題列點擊「+」進入。畫面切換為 A/B 並排：

| 欄位 | 說明 |
|------|------|
| A 欄（左） | 即時戰鬥數據，同步更新 |
| B 欄（右） | 選擇歷史戰報載入 |
| 頂部差值摘要 | 顯示總傷害和 DPS 的 A vs B 差異 |

點擊「−」退出比對模式，回到正常主視窗。

### 戰報管理

從右鍵選單「戰報管理」開啟。

| 操作 | 說明 |
|------|------|
| 篩選 | 依傷害下限或時長上限篩選戰報 |
| 全選 / 取消選擇 | 批次選取操作 |
| 上傳所選 | 將選取的戰報上傳到社群分析平台（條件：有效時長 ≤ 540 秒，傷害 ≥ 100,000） |
| 刪除 | 永久刪除選取的戰報 |

### 社群分析平台（ntedpsmeter.com）

線上平台 [ntedpsmeter.com](https://ntedpsmeter.com)，匯集社群戰鬥數據：

- **角色傷害排行榜** — Tier 分級（1min / 3min / 6min / 9min），顯示中位數 + 最大值 + 區間
- **編隊配對矩陣** — 視覺化角色搭配採用率（贊助者限定）
- **Connect 流程** — 貼上 Device ID 或 Analytics Token 即可在網頁查看個人數據
- **三語網站** — EN / 繁中 / 簡中
- **隱私友善** — 上傳前須明確同意，數據匿名化

| 頁面 | 條件 | 說明 |
|------|------|------|
| Rankings（排行榜） | 上傳過戰報 + Connect 綁定 | 各角色傷害中位數排行 |
| Compositions（編隊） | Sponsor + Connect 綁定 | 社群熱門角色組合與採用率 |
| Connect（綁定） | — | 輸入 Device ID 與 Analytics Token 綁定裝置，解鎖排行榜與編隊頁面 |

**Connect 流程：**
1. 從程式右鍵選單「複製 Device ID」取得裝置識別碼
2. 從右鍵選單「複製 Analytics Token」取得分析 Token（需 Sponsor）
3. 前往 ntedpsmeter.com/connect，貼上兩個值完成綁定

## 系統需求

- Windows 10 / 11
- [Npcap](https://npcap.com/) — 封包擷取驅動（安裝時勾選 WinPcap 相容模式）
- 以**系統管理員身份**執行

## 使用方式

1. 安裝 Npcap
2. 執行 `NTE_DPS_Meter.exe`（右鍵 → 以系統管理員身份執行）
3. 啟動遊戲，進入戰鬥後浮窗自動顯示 DPS
4. 從系統匣圖標可開啟主視窗、選擇網卡、設定快捷鍵

### 預設快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `F6` | 重置統計並存檔戰報 |
| `Alt + D` | 顯示/隱藏浮窗 |
| `Alt + E` | 顯示/隱藏主視窗（Pro） |

快捷鍵可在「快捷鍵設定」中自訂。

## 安全性說明

本工具僅被動讀取網路封包，原理等同 Wireshark：

- 不讀取、不修改遊戲記憶體
- 不注入 DLL 或 hook 遊戲進程
- 不發送任何封包至遊戲伺服器
- 不修改任何遊戲檔案

## 常見問題

**Q: 浮窗沒有反應？**
A: 確認已以系統管理員身份執行，且 Npcap 已正確安裝。

**Q: 開了加速器後抓不到封包？**
A: 從系統匣選擇「選擇網卡」，手動切換到正確的網路介面。

**Q: 顯示的傷害數值準確嗎？**
A: 數值直接來自遊戲客戶端發送的戰鬥封包，與遊戲內部計算一致。

**Q: 為什麼一直跳防毒偵測？**
A: 因為本程式尚未購買 EV 程式碼簽章憑證（價格高昂），Windows Defender 可能將其標記為不明程式。請將主程式（NTE_DPS_Meter.exe）及自動更新器（NTE_Updater.exe）加入排除清單。未來贊助資金足夠時會購買 EV 簽章解決此問題。

桌面右鍵 → 顯示設定 → 隱私權與安全性 → Windows 安全性 → 病毒與威脅防護 → 管理設定 → 向下滾動找到「排除項目」→ 新增排除項目 → 新增排除範圍 → 資料夾 → 選擇程式所在目錄
